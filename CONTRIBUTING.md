# Contributing to Internet Scanner

Thank you for your interest in contributing to **Internet Scanner**! 🎉

Internet Scanner is an open-source network discovery and analysis tool built with Python. It uses technologies such as **Scapy** and **Nmap** to discover devices, analyze networks, and provide useful security insights.

All contributions are welcome, whether you are fixing bugs, improving the interface, adding features, or improving documentation.

---

## Ways You Can Contribute

There are many ways to help improve the project:

### 🐛 Report Bugs

Found something that does not work correctly?

Please open an issue and include:

* What happened
* Steps to reproduce the problem
* Your operating system
* Python version
* Any error messages or screenshots

---

### ✨ Suggest Features

Have an idea that could improve Internet Scanner?

Examples:

* Better scanning methods
* New security checks
* Improved GUI features
* Performance improvements
* Support for more platforms

Open a feature request issue and explain:

* What the feature does
* Why it would be useful
* Any ideas for implementation

---

### 💻 Code Contributions

You can help by:

* Fixing bugs
* Improving existing code
* Adding new functionality
* Optimizing performance
* Improving compatibility

---

### 📚 Documentation

Documentation improvements are always appreciated.

You can help by:

* Fixing unclear explanations
* Adding examples
* Improving setup instructions
* Adding screenshots

# Getting Started

## Requirements

Before contributing, make sure you have:

* Python 3.10 or newer
* Nmap installed
* Npcap installed (Windows users)
* Git

---

## Setting Up the Project

Clone the repository:

```bash
git clone https://github.com/Fa1dz/Internet-Scanner.git
```

Move into the project folder:

```bash
cd Internet-Scanner
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate it:

Windows:

```bash
.venv\Scripts\activate
```

Linux/macOS:

```bash
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python Internet_Scanner.py
```

---

# Making Changes

Create a new branch for your changes:

```bash
git checkout -b feature-name
```

Examples:

```bash
git checkout -b add-dark-mode
```

```bash
git checkout -b improve-scanner-speed
```

Make your changes, test them locally, and commit:

```bash
git add .
git commit -m "Add dark mode support"
```

Push your branch:

```bash
git push origin feature-name
```

Then open a Pull Request.

---

# Pull Request Guidelines

Before submitting a Pull Request:

✅ Test your changes
✅ Explain what you changed
✅ Keep commits clear and descriptive
✅ Avoid unrelated changes
✅ Update documentation if needed

A good Pull Request should explain:

* What problem it solves
* What was changed
* How it was tested

---

# Good First Contributions

If you are new to the project, these are good places to start:

* Add dark mode support
* Improve GUI layout
* Add more export formats
* Improve error handling
* Add unit tests
* Improve documentation
* Add Linux/macOS compatibility
* Add new network analysis features

---

# Code Style

Please try to keep the code:

* Clean and readable
* Properly commented when needed
* Consistent with the existing project style
* Easy for other contributors to understand

---

# Security Notice

Internet Scanner should only be used on networks and devices you own or have permission to test.

Unauthorized scanning of networks can be illegal.

If you discover a security issue in the project, please report it privately instead of opening a public issue.
